\section*{System Demonstration}
\label{sec:demo}

We provide a live web demo that lets users interact with \ours through a browser.
A FastAPI backend runs the full CLIP + BLIP + Flan-T5 pipeline and communicates with the front end over WebSockets.
The interface splits into a video panel (live feed plus SKM filmstrip with importance scores) and a chat panel where answers appear in under 200\,ms on GPU with scope tags and latency indicators.
Users can stream from their webcam, upload a video file, or choose from pre-loaded sample clips.

A dedicated \emph{workshop presentation mode} enlarges all UI elements for projector visibility and offers one-click sample questions that step through all three temporal scopes, letting the presenter walk the audience through the pipeline interactively.
The demo runs on GPU or CPU (1--3\,s per query on CPU) and ships with a Docker configuration for one-command startup.
Code and demo will be released upon acceptance.
